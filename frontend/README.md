# 🌈 Frontend application

---

## 🏗️ Setup Local Development

1. **Install dependencies**:
   ```bash
   pnpm install
   ```
2. **Run application**
   ```bash
   pnpm run dev
   ```

## 🖍️ Code formatting
There is a script that fixes code styles. In case it can't fix code automatically, it shows errors
```bash
  pnpm run lint:fix
```
